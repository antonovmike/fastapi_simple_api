# fastapi_simple_api
FastAPI essencials: Creating simple API

Simple Web API REST app


![Simple API](https://github.com/antonovmike/fastapi_simple_api/blob/main/Screenshot.png)

```bash
pip install fastapi
pip install "uvicorn[standard]"
```

START:
```bash
uvicorn main:app --reload
```
Open

http://127.0.0.1:8000/
